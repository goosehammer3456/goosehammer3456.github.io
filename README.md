# goosehammer3456.github.io

Personal "link in bio" page for Bruce Johnson — Sales Engineer Manager at Delinea.

**Live site:** [goosehammer3456.github.io](https://goosehammer3456.github.io)

## Features

- Clean, modern single-page design with Delinea brand green theming
- Dark/light mode toggle with persistent user preference (localStorage)
- Animated page load entrances (staggered fade-up)
- Hover effects on link cards (lift, glow shadow)
- SVG favicon with initials
- Fully responsive — works on mobile, tablet, and desktop

## Tech Stack

- Single `index.html` file — no frameworks, no build step
- Pure HTML, CSS, and vanilla JavaScript
- Hosted on GitHub Pages

## Local Development

Open `index.html` in a browser. That's it.

## Deployment

Pushing to `main` automatically deploys via GitHub Pages.
